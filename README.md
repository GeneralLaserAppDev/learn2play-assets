# Learn2Play — public brand assets

Public, CDN-served brand assets for Learn2Play (logos used in transactional
emails, website, store listings, etc.).

These files are intentionally public so they can be referenced from HTML emails
and other external surfaces.

## Logo

- `logo.png` — full-resolution app logo (1024×1024)
- `email/logo.png` — email-optimized copy (256×256)

Recommended embed URL (jsDelivr CDN — note the branch is `master`, not `main`):

```
https://cdn.jsdelivr.net/gh/GeneralLaserAppDev/learn2play-assets@master/email/logo.png
```

## Store badges

Official App Store / Google Play badges (black, 120 px tall, 3× retina for a
~40 px display height), localized in de / en / bg. Used in transactional emails:

- `email/badges/appstore-{de,en,bg}.png` — "Download on the App Store"
- `email/badges/googleplay-{de,en,bg}.png` — "Get it on Google Play"

```
https://cdn.jsdelivr.net/gh/GeneralLaserAppDev/learn2play-assets@master/email/badges/appstore-de.png
https://cdn.jsdelivr.net/gh/GeneralLaserAppDev/learn2play-assets@master/email/badges/googleplay-de.png
```
